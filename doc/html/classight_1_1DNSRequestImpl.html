Detailed Description

Implementation of DNSRequest.

This is the internal object thanks to which DNSRequest is movable. Of course, DNSRequestImpl is not movable, since its address is passed to one of the many evnds delayed requests functions.

Remarks
You should not use this class directly, use DNSRequest instead.
