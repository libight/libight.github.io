Detailed Description

DNS Resolver object.

This object can be used to construct specific DNS resolvers that differ from the default DNS resolver of libight.

Once a custom resolve is created, one can use it to issue DNS requests.

In other words, to use the default DNS resolver, one does not need to create an instance of this object.

For example: 

auto reso = ight::DNSResolver(ight::DNSSettings()
        .set_nameserver("8.8.8.8")
        .set_timeout(1.0)
        .set_attempts(1));

auto r2 = reso.request("REVERSE_AAAA", "2001:858:2:2:aabb:0:563b:1e28",
        [](ight::DNSResponse&& response) {
    // Process the response
});
