Detailed Description

DNS response.

The constructor of this class receives the fields returned by evdns and converts them in a format suitable to compile OONI's reports.

You should not construct this class directly. Instead, this is the object that DNSRequest's callback provides you.

See also
DNSRequest for example usage.
